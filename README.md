#ImageCap


## 简介
一款集成网页图片批量下载和网页截全屏功能的Chrome插件，界面设计简简洁，操作简单。

## 功能
1. 网页智能图片采集
2. 网页截屏
3. 支持大图下载（目前支持新浪微博, 新增QZONE）
4. 网页截屏和图片标注
5. 超长网页截屏
